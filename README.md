## 快速开始

### 安装依赖

此项目提供的解析器工具链可能需要部分依赖（Agent 在后台调用解析工具时使用）：

```bash
pip install -r requirements.txt
```

可选依赖：

```bash
# 向量搜索
pip install numpy faiss-cpu sentence-transformers

# PDF 解析
pip install PyPDF2

# LLM 问答
pip install openai
```

### 在 Agent 中启用 Skill（推荐）

将本目录作为 Skill 包或普通 Workspace 加载到你喜欢的 Agent 工具（如 **OpenClaw** 或 **Claude Code**）中，确保引擎能够读取到本工程结构（尤为重要的是 `prompts/` 目录内的引擎执行流）。

触发无需任何命令行菜单，全程自然语言交互：

- 新建：帮我蒸馏人生 / 用我的聊天记录生成人生画像
- 更新：我有新聊天记录，更新这个 skill
- 纠偏：这不像我 / 你理解错了

## 使用

### Agent 对话自动化触发

摒弃了繁琐的命令行解析流程，在你的 Agent 中直接唤起。一切调度交由大模型原生理解决定。

**推荐唤起话术：**

- "帮我蒸馏人生，这是我的聊天记录导出文件 /path/to/data"
- "用我上传的日记得出我的人生底层代码体系"
- "昨天和老板吵架了，这是我的回怼记录，请增量萃取更新进入我的人生 Skill"
- "这不像我，我绝对不会借钱给不熟的人，请修正 Life Model"

**Agent 自动执行链路：**

1. **材料获取**：识别你提供的路径或粘贴文本，按需调用后台 `tools/` 解析器（微信/邮件/文档）。
2. **分析与挖掘**：调度 `life_analyzer.md` 与 `persona_analyzer.md`，使用统一证据协议提取要素。
3. **架构与合并**：调度 `life_builder.md` 对新项目进行立卷，或调度 `updater.md`/`merger.md` 对现有档案进行无冲突或纠偏式的增量合并。
4. **质量验证**：执行 `validator.md` 中定义的 Gate A-D 四道核设闸门并处理不合规打回。
5. **交付落盘**：最终生成或更新结构化内容至 `lives/{slug}`，并自动维护 `CHANGELOG.md`。

## 效果示例

### 1) 朋友找你借钱（判断亲密关系 + 自身资金）

输入 A（关系一般）：
`很久不联系的朋友突然发消息：借我 2 万，月底还你。`

输出：
`（不回复）`
`打开app store，下载某小满`
`打开某音， 放心借`

输入 B（关系较好，但手头紧）：
`关系不错的朋友找你借 2 万，你本月现金流紧张。`

输出：
`我理解你现在着急，但是我真没钱。`
`但我这段时间资金卡得很紧，确实拿不出这笔。`

### 2) 想跟某个女生表白（判断信任程度 + 当前占用）

输入 A（关系一般）：
`我想跟 XXX 表白，有戏吗？`

输出：
`舔狗舔狗，舔到最后一无所有`

输入 B（关系亲密）：
`我想跟 XXX 表白，有戏吗？`

输出：
` 有戏`

## 版本与纠偏管理

当分身表现出的行为发生偏差，或你需要指定分身回到过去某个状态：

**通过对话反馈即可：**

- "查看 zhangsan 的版本历史记录"
- "对比下现在的我跟 v1 版本的我，在风险承受偏好上有什么不同"
- "刚才你强行把我标成了讨好型人格，这不像我，我只是出于效率懒得争辩！请触发 CORRECT 更新修正！"
- "回滚到上个版本 v2，最近这批语料污染太严重了"

（Agent 会自动调度对应逻辑及 Prompt 里的机制完成降级与回退。）

## 功能特性

- **双结构输出**：Life Model (底层人设基建) + Voice Persona (行为与表达模式)。
- **纯对话触发流**：完全拥抱 Agent Native，无需繁琐的 Python CLI 入口。
- **统一证据链追溯**：每个决策推断自带置信度门栏 `(★★★)` 和证据引用 `[E序号]`。
- **关系分级分析**：S+/S/A/B/C/D 网络社交降维打击。
- **强制合规守门**：自带 Validator 质量核查漏斗阻断隐私与幻觉外溢。
- **高级增量控制**：支持带状态标记的合并以及用户高优先级显式纠偏。

## 注意事项

- **数据丰富度**：聊天记录质量决定还原度，多场景且包含矛盾处理的数据远优于单一的日常敷衍记录。
- **谨慎脑补**：如果材料极度不足，分身应当果断回答 "不清楚/没遇到过"，而非产生大模型原生的烂好人幻觉。
- **隐私脱敏提示**：引擎自带脱敏闸门，但也建议用户不要直接输入极度私密的账密层敏感信息。
